# Руководство по созданию модов для Minecraft 1.20.1 Forge

Полный гайд по разработке модов на Minecraft 1.20.1 с использованием Forge. Подходит как для новичков, так и для опытных разработчиков.

## 📋 О содержании

Это руководство охватывает все аспекты создания модов — от базовой настройки до продвинутых техник. Материал структурирован по принципу «от простого к сложному», что позволяет постепенно погружаться в мир моддинга.

## 🚀 Быстрый старт

### Требования
- Базовое знание хотя-бы одного ООП языка программирования со статической типизацией (предпочтительно Java)
- Понимание основ ООП (объектно-ориентированного программирования)

### Важное замечание! 

Вы можете добавить пару своих блоков или крафтов даже **БЕЗ** знаний в программировании.

### Время на изучение
- **Базовый уровень** (предметы, блоки, крафты): 1 час — 1 день
- **Продвинутый уровень**: пока не знаю (в разработке)
- **Экспертный уровень**: пока не знаю (в разработке)

## 🛠️ Необходимые инструменты

- **IntelliJ IDEA** — основная IDE для разработки
- **Forge MDK 1.20.1** — набор для разработки модов
- **Java 17** — требуемая версия JDK
