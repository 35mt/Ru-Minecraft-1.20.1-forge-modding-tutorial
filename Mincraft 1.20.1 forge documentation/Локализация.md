для того чтобы добавить локализацию в свой мод нужно перейти в каталог "lang" в resources, однако если у вас ещё нет подготовленной структуры в этой папке то прочитайте этот небольшой гайд на создание структуры в resouces: [[Создание структуры папки resources]]
в каталоге "lang" нужно создать .json файл с названием, соответствующем коду языка - для русского перевода это будет файл ru_ru.json, а для английского - en_us.json. код языка должен быть написан строчными буквами, а так-же иметь нижнее подчёркивание вместо тире.

теперь зайдите в созданный файл и (как и в любом .json файле) создайте фигурные скобки если их ещё нет - всё содержимое таких файлов должно быть внутри них. далее все последующие записи в этом файле будут представлять собой пару - "ключ - значение" в формате "ключ" : "значение" отделяемые через запятую, например:
```json
{  
  "item.myfirstmod.item1" : "Предмет 1",
  "item.myfirstmod.item2" : "Предмет 2"  
}
```
