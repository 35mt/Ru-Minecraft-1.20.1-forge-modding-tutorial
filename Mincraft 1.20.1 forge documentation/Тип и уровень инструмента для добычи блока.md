если у вас ещё нет готовой структуры папки resources, то создайте её по этому [[Создание структуры папки resources|гайду]].
для этого сначала перейдите в **resouces.data.minecraft.tags.blocks** и создайте файл **needs_iron_tool.json** затем перейдите в него и создайте простую структуру:
```json
{
	"values": [
		"ваш mod:код блока",
		"myfirstmod:block1" // пример
	]
}
```
