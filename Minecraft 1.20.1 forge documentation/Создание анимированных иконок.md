Для создания анимированных иконок предметов нужно иметь спрайт-шит с анимацией. Далее мы закидываем спрайт шит в папку **resources.assets.MOD_ID.textures.item**, а затем создайте файл .mcmeta с названием, полностью аналогичным названию спрайт-шита (включая расширение!) Например если ваш спрайт-шит называется "picture1.png", то файл .mcmeta будет иметь название "picture1.png.mcmeta". 

**ВАЖНО!** Чтобы названия совпадали.
## Разделение на кадры

Далее перейдите в этот файл и создайте там такую структуру:
```mcmeta
{  
  "animation": {  
    "frametime": 10,  
    "frames": [0, 1, 2, 0, 1, 2]  
  }  
}
```

В этом файле ваш спрайт-шит разделяется на кадры - 
- "frametime" : - Здесь указывается время задержания 1 кадра в тиках, на например в данном примере один кадр будет длится пол секунды.
- "frames" : - Здесь указывается порядок кадров - индексация начинается с 0, слева на право. Желательно указывать последовательность кадров 2 раза для надёжности
## Привязка текстуры к предмету

Теперь как и при создании обычной модели - перейдите в папку **resources.assests.MOD_ID.models.item** а затем создайте .json файл с названием, которое идеально совпадает с кодом предмета.
Затем перейдите в этот файл и создайте там такую структуру:
```json
{  
  "parent" : "item/generated",  
  "textures" : {  
    "layer0" : "MOD_ID:item/ITEM_ID"  
  }  
}
```
